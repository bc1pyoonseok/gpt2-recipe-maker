
# GPT2 recipe-maker
[![Run on Ainize](https://ainize.ai/images/run_on_ainize_button.svg)](https://ainize.web.app/redirect?git_repo=https://github.com/ha-mulan/gpt2-recipe-maker)



### About

	it generates recipe related to your input keyword.
	first, input your keyword
	second, wait for result

### How to use

	three ways to use Gpt2 recipe-maker
    	1.  CLI
    	2.  Swagger
    	3.  Demo

### GET parameter

    keyword: input your keyword than you will check out the recipe related to keyword

### Output format

    generated text


##  *With CLI*

### Input example

    curl -X GET "https://master-gpt2-recipe-maker-ha-mulan.endpoint.ainize.ai/api/?food=cake" -H "accept: string"


### Output example

      cake:

      1 red bell pepper, diced
      4 cloves garlic, minced
      1 teaspoon ground coriander
      1/4 teaspoon ground cayenne pepper
      1/4 cup olive oil
      salt and pepper to taste
      1 pound ground beef
      1/2 cup chopped parsley
      1 onion, chopped
      4 sprigs fresh parsley
      1 tablespoon Worcestershire sauce
